# Challenge Name

Spring will be here soon...

# Author

Ryan, John or James?

# Description

I was clearing out my phone storage when I stumbled upon this photo.

The only thing I remember is that I took it shortly after visiting a temple.

Can you determine the name of the street where this photo was taken and the nearest metro station?

Flag format: LNC26{STATION_STREET}

# Solution

Given an image, reverse image the lamp post. Location is in Japan, Haedong Yonggungsa Temple. 

We need the nearest metro station and street of where the picture is taken. Nearest station would be OSIRIA. 

There is a turn in the picture and a green fence so look around and get the street name YONGGUNG-GIL

# Flag

LNC26{OSIRIA_YONGGUNG-GIL}
