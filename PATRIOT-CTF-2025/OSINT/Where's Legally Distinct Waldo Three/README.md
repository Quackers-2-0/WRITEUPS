# Challenge Name

Where's Legally Distinct Waldo Three

# Author

Sid Kumar

# Description

Find the building where Waldo is. The flag is the building from which this photograph was taken. The full building name is the flag. Seperate spaces with underscores.

The building name should have underscores instead of slashes or spaces.

This OSINT challenge has 10 maximum attempts.

# Solution

A pond was visible in the image, which matched Mason Pond on the George Mason University campus. Surveying nearby structures revealed the Center for the Arts complex. The challenge required the full building name, resulting in Center for the Arts Concert Hall.
