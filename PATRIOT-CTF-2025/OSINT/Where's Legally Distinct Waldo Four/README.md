# Challenge Name

Where's Legally Distinct Waldo Four

# Author

Sid Kumar

# Description

Find the building where Waldo is. The flag is the building from which this photograph was taken. The full building name is the flag. Seperate spaces with underscores.

This OSINT challenge has 5 maximum attempts.

# Solution

The image showed two adjacent buildings separated by a wall. Identifying Fenwick Library as a nearby landmark allowed orientation of the image. Using Google Earth and analyzing the building angle, Krug Hall was determined to be the correct location.
