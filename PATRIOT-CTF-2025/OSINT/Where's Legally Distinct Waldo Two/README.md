# Challenge Name

Where's Legally Distinct Waldo Two

# Author

Sid Kumar

# Description

Find the building where Waldo is. The flag is the building from which this photograph was taken. The full building name is the flag. Seperate spaces with underscores.

This OSINT challenge has 10 maximum attempts.

# Solution

The image contained partial parking lot identifiers beginning with the letter 'F'. Reviewing parking lots at George Mason University led to Finley Lot. From the image angle and surrounding structures, the nearby building was confirmed to be Thompson Hall.
